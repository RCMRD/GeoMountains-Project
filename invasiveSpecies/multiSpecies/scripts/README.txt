The file labelled "biomod2_multi_species_modelling_1.r" is an R script that models Opuntia and Acacia Invasive species. It uses the selected variables, obtained when running the single species models, using climatic variables only.


The file labelled "biomod2_multi_species_modelling_2.r" is an R script that models Opuntia and Acacia Invasive species. It uses the selected variables, obtained when running the single species models, using physical variables only.


The file labelled "biomod2_multi_species_modelling_3.r" is an R script that models Opuntia and Acacia Invasive species. It uses the selected variables, obtained when running the single species models, using both bioclimatic and biophysical variables.
