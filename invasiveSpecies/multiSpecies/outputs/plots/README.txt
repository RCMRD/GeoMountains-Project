The outputs result from the Level plot function.
There exist the
a) climatic plot - resulting from running the multispecies code with bioclimatic variables only.
b) physical plot -  resulting from running the multispecies code with biophysical variables only.
c) climatic-physical - resulting from running the multispecies code with both bioclimatic and biophysical variables.

The MEAN was used in all instances. Additionally, COMMITTEE AVERAGING (CA) was applied for the instance of combined variables.
